## 📋 Estructura de cada caso

Cada auditoría sigue este formato:

- **Contexto** — Entorno y antecedentes
- **Hallazgos** — Vulnerabilidades o problemas identificados
- **Análisis** — Proceso de investigación y herramientas usadas
- **Solución** — Pasos aplicados para resolver
- **Lecciones aprendidas** — Qué enseña este caso sobre seguridad
